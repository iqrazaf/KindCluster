To create a Kind-Cluster, perform a load test and collect all the data (present in a data folder) 
through a Linux Machine, the following are the Prerequisites:

Host requirements
- Linux recommended
- 8+ CPU cores, 16+ GB RAM recommended (minimum 4 cores / 8 GB)
- Docker must be running

Required tools (versions are suggestions)
- Docker (20+)
- kubectl (1.26+)
- kind (0.20+)
- helm (3.10+)
- istioctl (1.18+ or match your Istio manifests)
- jq (optional, but useful)

Verify tools:
docker --version
kubectl version --client
kind version
helm version
istioctl version

For Python code running:
- Python 3.9+ recommended

Install packages:
bash
pip install optuna pandas numpy matplotlib networkx
